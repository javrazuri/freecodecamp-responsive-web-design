Introducción a Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Su filosofía enfatiza la legibilidad del código, lo que lo hace ideal para principiantes.

Fue creado por Guido van Rossum y lanzado por primera vez en 1991. Desde entonces, ha crecido en popularidad debido a su simplicidad y versatilidad.

print("¡Hola, mundo!")

Este sencillo comando muestra texto en la consola, y es tradicionalmente el primer programa que se escribe al aprender un nuevo lenguaje.

Sintaxis Básica

Python utiliza la indentación (espacios o tabulaciones) para definir bloques de código, en lugar de llaves {} como en otros lenguajes.

if 5 > 2: print("Cinco es mayor que dos")

La indentación es obligatoria. Si no se respeta, Python arrojará un error.

  • Las líneas no terminan con punto y coma.
  • Los comentarios se escriben con #.
  • Las cadenas pueden usar comillas simples o dobles.
Tipos de Datos

Python tiene tipos de datos dinámicos. No necesitas declarar el tipo de una variable al crearla.

x = 5 # int y = "Hola" # str z = True # bool

Algunos tipos de datos comunes incluyen:

  • int - Números enteros
  • float - Números decimales
  • str - Cadenas de texto
  • list - Listas (arreglos modificables)
  • bool - Booleanos (True o False)
Estructuras de Control

Permiten controlar el flujo de ejecución del programa.

Ejemplo de estructura if-else:

edad = 18 if edad >= 18: print("Eres mayor de edad") else: print("Eres menor de edad")

Ejemplo de bucle for:

for i in range(5): print(i)
  • El bucle for itera sobre secuencias.
  • El bucle while se repite mientras una condición sea verdadera.
Funciones en Python

Una función es un bloque de código reutilizable que realiza una tarea específica.

def saludar(nombre): return "Hola, " + nombre mensaje = saludar("Ana") print(mensaje)

Características clave:

  • Se definen con la palabra clave def.
  • Pueden recibir parámetros y devolver valores.
  • Las funciones mejoran la modularidad y reutilización del código.

Python también permite funciones anónimas con lambda:

doble = lambda x: x * 2 print(doble(5)) # Imprime 10
